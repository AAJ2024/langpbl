# Unsloth Complete Installation Guide for New Laptop

## System Requirements
- Windows 10/11
- NVIDIA GPU with CUDA support
- At least 16GB RAM recommended
- 20GB+ free disk space

---

## Step 1: Check Python Version

First, check what Python versions you have installed:

```bash
py --list
```

**IMPORTANT:** Unsloth requires Python 3.9 to 3.13 (NOT 3.14)

If you see Python 3.14, you need to use Python 3.13 instead.

---

## Step 2: Set Python 3.13 as Default (if needed)

If you have Python 3.14, switch to 3.13:

```bash
# Check current version
python --version

# If it shows 3.14, use py launcher to set 3.13
py -3.13
```

To make Python 3.13 default, you can:
1. Go to Windows Settings > Apps > App execution aliases
2. Turn OFF Python 3.14
3. Keep Python 3.13 ON

Or add Python 3.13 to PATH manually:
- Right-click "This PC" > Properties > Advanced System Settings
- Environment Variables > System Variables > Path
- Move Python 3.13 path to the top

---

## Step 3: Upgrade pip

```bash
python.exe -m pip install --upgrade pip
```

---

## Step 4: Install PyTorch with CUDA Support

**CRITICAL:** Install PyTorch BEFORE Unsloth

```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

This will download about 2.8GB, so be patient.

---

## Step 5: Verify CUDA Installation

```bash
python -c "import torch; print(f'CUDA available: {torch.cuda.is_available()}')"
python -c "import torch; print(f'CUDA version: {torch.version.cuda}')"
python -c "import torch; print(f'GPU Name: {torch.cuda.get_device_name(0)}')"
```

You should see:
- `CUDA available: True`
- `CUDA version: 11.8` (or similar)
- Your GPU name

---

## Step 6: Install Unsloth

```bash
pip install "unsloth[colab-new] @ git+https://github.com/unslothai/unsloth.git"
```

This will take several minutes and install many dependencies.

---

## Step 7: Install Additional Dependencies (if needed)

If you get missing dependency errors, install these:

```bash
pip install --no-deps trl peft accelerate bitsandbytes datasets
```

---

## Step 8: Verify Unsloth Installation

```bash
python -c "import unsloth; print('Unsloth installed successfully!')"
```

---

## Troubleshooting

### Problem: "Package 'unsloth' requires a different Python"
**Solution:** You're using Python 3.14. Switch to Python 3.13 (see Step 2)

### Problem: "CUDA available: False"
**Solution:** 
1. Make sure you have an NVIDIA GPU
2. Install NVIDIA drivers from nvidia.com
3. Reinstall PyTorch with CUDA: `pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118`

### Problem: "ModuleNotFoundError: No module named 'unsloth'"
**Solution:** 
1. Make sure you're using the correct Python version
2. Reinstall: `pip install "unsloth[colab-new] @ git+https://github.com/unslothai/unsloth.git"`

### Problem: Missing dependencies errors
**Solution:** 
```bash
pip install accelerate transformers datasets trl peft bitsandbytes
```
